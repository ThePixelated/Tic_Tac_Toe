# Tic Tac Toe
This is a simple game of tic tac toe made with C++

![image](https://github.com/ThePixelated/Tic_Tac_Toe/assets/44131061/ce046cba-70d6-41de-bef1-c16230e51ca6)

### Keys configuration
- **W**: upward
- **A**: downwards
- **S**: left
- **D**: right
- **Space** or **Enter** to select cell
- **E**: exit
